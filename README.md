# TOTAL MOBILIZATION

A single-file browser RTS — defend a slice of Earth from an autodrone invasion organized by atmosphere layer. Build industry, research tech, and field a planetary defense.

**Play:** open the site URL (it forwards to `title.html`). The whole game is one self-contained HTML file — no install, no build step.

Best in a desktop browser (Chrome/Edge/Firefox). Music streams via the YouTube player; everything else is rendered live on a `<canvas>`.
